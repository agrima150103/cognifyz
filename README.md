# Cognifyz Full Stack Project

## Overview
A full stack web application built as part of the Cognifyz Full Stack Development assessment.

## Features
- User Authentication (Login/Signup)
- Secure Backend APIs
- Database Integration
- Responsive UI
- CRUD Functionality
- Deployed Live Application

## Tech Stack
Frontend:
- React.js
- HTML/CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Installation
Install dependencies:
```bash
npm install
```

Run project:
```bash
npm start
```
